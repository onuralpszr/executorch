Class LlmGenerationConfig

public class LlmGenerationConfig

    Configuration class for controlling text generation parameters in LLM operations.

    This class provides settings for text generation behavior including output formatting, generation limits, and sampling parameters. Instances should be created using the method and the fluent builder pattern.
